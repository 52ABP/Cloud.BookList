ABP是“ASP.NET Boilerplate Project (ASP.NET样板项目)”的简称。 ASP.NET Boilerplate是一个用最佳实践和流行技术开发现代WEB应用程序的新起点，它旨在成为一个通用的WEB应用程序基础框架和项目模板。 ASP.NET Boilerplate 基于DDD的经典分层架构思想，实现了众多DDD的概念（但没有实现所有DDD的概念）。 ABP的官方网站：http://www.aspnetboilerplate.com ABP在Github上的开源项目：https://github.com/aspnetboilerplate

ASP.NET Boilerplate框架旨在独立于任何数据库模式，并尽可能通用。因此，它留下了一些抽象和可选的概念（如审计日志记录，权限和设置存储），这需要一些数据存储。 Module Zero实现了ASP.NET Boilerplate框架的所有基本概念，例如租户管理（多租户），角色管理，用户管理，授权（权限管理），设置管理，语言管理，审计日志记录等。 Module-Zero定义实体并实现域逻辑（域层）并将应用程序和表示层留给您。

Module Zero基于 Microsoft ASP.NET Core Identity它扩展了用户和角色管理器，并使用通用存储库实现用户和角色存储。
